# back2vibing

The missing layer for your AI workflow. Run two, three, or even a hundred agents in parallel without breaking your flow.

## Installation

### Install via Homebrew

```bash
brew tap builtby-win/back2vibing https://github.com/builtby-win/back2vibing
brew install back2vibing
```

Or install the latest release directly:

```bash
brew install --cask back2vibing --github-token <your-github-token>
```

### Install Nightly Build

Get the latest development version with the newest features:

```bash
brew tap builtby-win/back2vibing https://github.com/builtby-win/back2vibing --branch nightly
brew install back2vibing@nightly
```
